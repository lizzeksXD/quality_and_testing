Мини-песочница №1 — MVC в Flask

Простая демонстрация разделения контроллера, модели и представления.

Это небольшое приложение на Flask, где:

  • Model — доменные классы и репозиторий (app/models)
  • Controller — thin-контроллеры (app/controllers)
  • View — Jinja2 шаблоны (app/views)

В примере реализована простая работа с книгами и заглушечная аутентификация для демонстрации редиректов.

{% if current_user.is_authenticated %} Выполнен вход как: {{ current_user.username }} Выйти {% else %} Аутентификация {% endif %} Перейти к книгам
